# Focus & Cafe Roulette

勉強と休憩を「ルーレット」で決めて集中するための、個人用デスクトップアプリ。
Python + Streamlit 製。設定・履歴・その日の状態はすべてローカルのファイルに保存され、
**閉じて開き直しても状態が復元**されます。

---

## 技術スタック
- **言語**: Python（画面の一部は埋め込みの HTML / CSS / JavaScript）
- **フレームワーク**: Streamlit（UI。リアルタイムのカウントダウン等は `components.html()` で JS を差し込み）
- **ライブラリ**: `streamlit`, `pandas`（CSV読み書き）＋ 標準ライブラリ（json / os / datetime / random / base64 など）
- **データ保存**: DBは使わず、`data/` フォルダ内の JSON / CSV ファイル

## 起動方法
1. Python と必要ライブラリを用意（初回のみ）
   ```
   pip install streamlit pandas
   ```
2. `project` フォルダ内で起動
   - かんたん: `run.bat` をダブルクリック（古いキャッシュを掃除してから起動します）
   - または手動:
     ```
     cd project
     streamlit run app.py
     ```
   ブラウザが開きます（http://localhost:8501）。停止はターミナルで `Ctrl + C`。

## 主な機能

### ダッシュボード（1ページ・左右2分割）
- **左：🎲 ルーレット ＆ 残り作業可能時間**
  - 「カフェルーレットを回す！」で、勉強／気分転換のお題を抽選 → 「集中モードへ！」で開始
  - 連続で気分転換しないよう、気分転換の次は強制的に勉強になる
  - 目標（複数可）までの「残り作業可能時間（日数 × 1日の作業時間）」をカード表示。高さは自動でそろう
- **右：📊 今日の進捗**
  - **今日まだ勉強できる時間**を「◯時間◯分◯秒」で**1秒ごとにカウントダウン**（後述の曜日別時間帯で計算）
  - 今日の目標勉強時間（1日1回だけ確定。確定後は変更不可・大きく表示）
  - 進捗ドーナツ（%）・今日の勉強/気分転換の合計

### 🔴 今日絶対やる（優先カテゴリ）
- 「今日絶対やる」リストに項目があると、**勉強の抽選はそのリストだけ**から出る
- 中身が空になると、自動で通常勉強の抽選に戻る
- タスク終了後の「ふりかえり」で「終わった？消す？」を質問。消すと回答すると**自動でリストから削除**

### 集中モード（タイマー）
- 気分転換は残り時間が見えるカウントダウン、勉強は「規定時間を達成するまで終了ボタンが出ない」仕様
- 終了時刻になるとアラーム（ビープ＋画面演出）
- 没入(コンパクト)モード（背景メインの最小表示）
- 短い課題（既定分以下）は途中で「修了してもう1回抽選」も可能
- 終了時刻は `start_time`(Unix秒)基準なので、閉じて開き直してもカウント継続

### ふりかえり / SOS
- ふりかえり: やったこと・進捗度・集中度・満足度を任意で記録 → 履歴CSVへ
- SOS（集中切れ）: 5分タイマー＋経過後アラーム

### ゲーム解放
- 「勉強時間が目標以上」かつ「20時以降（〜深夜3時）」を満たすと、気分転換に本物のゲームが追加される

### 編集モード（サイドバーの「🛠️ 編集モード」）
- **ルーレット項目**: 🔴今日絶対やる / 📘通常勉強 / 🔥重点(3倍) / ☕気分転換 を個別に追加・編集・削除
  - 各項目に「有効/無効」チェック（消さずに抽選から除外）
  - 削除は「世代キー方式」で、押した行が確実に消える
  - 追加後は入力欄に自動でフォーカス（連続入力しやすい）
- **目標物**: 名前・日付・1日の作業時間を可変個（最大6個）で設定
- **勉強タスクの所要時間**: 下限30分固定、上限はスライダー/±/直接入力（10刻み）
- **🕐 勉強できる時間帯（曜日別・複数OK）**: 曜日ごとに勉強可能な時間帯を複数登録（大学・食事は除いて登録）。「今日まだ勉強できる時間」に反映
- **背景**: カフェ画像／黒画面／自分でアップロード（履歴から再選択も可）
- **環境音**: カフェ/雑踏/波/無音、または自分のYouTube URL（曜日別…ではなく自由に再生）

### サイドバー
- 🎧 環境音コントロール / 🖼️ 背景 / 🔄 モード切替 を折りたたみ式トグルで表示
- 黒い余白クリックで閉じる／左端の縦帯（ホバーで光る）で開く

## 保存されるデータ（`data/` 内）
| ファイル | 内容 |
|----------|------|
| `settings.json` | 設定全般：目標、各リスト（通常勉強/重点/気分転換/今日絶対やる）、有効無効、勉強時間の上限、曜日別の勉強可能時間帯、背景、環境音URL など |
| `activity_log.csv` | 履歴。1タスクごとに 日付/カテゴリ/内容/BGM/経過時間/やったこと/進捗度/集中度/満足度/メモ を記録（pandasで毎回全件書き直し＝列追加でも壊れない） |
| `session_state.json` | その日の状態：現在ページ・実行中タスク・タイマー開始時刻・今日の勉強/気分転換の合計・目標時間と確定フラグ・抽選結果 など |

> ※ `data/` のパスは `config.py` が「自分(config.py)の場所」基準で決めるので、
>   どのフォルダから起動しても保存先がブレません。

## 永続化のしくみ
- 値が変わるたびに `state.commit_rerun()`（= `save_session()` してから `st.rerun()`）で `session_state.json` に保存
- 設定変更は即 `save_settings()` で `settings.json` に保存
- 起動時 `init_session()` で復元
  - `target_date`（保存した日付）が**今日と同じ**なら全キー復元
  - **日付が変わっていたら** `target_locked` / `target_value(=180)` / `study_time_total` / `refresh_time_total` のみリセットし、他（ページ・実行中タスク等）は復元
- タイマーは `sessionStorage` を使わず `start_time` を基準にするため、再起動でもカウントが継続

## メモ
- データベース不要・ファイルだけのシンプル構成（個人用ツール向け）
- 環境音は YouTube 前提（Apple Music / Spotify は埋め込み制約のため非対応）
- 古いキャッシュ（`__pycache__`）が残ると更新が反映されないことがあるため、迷ったら `run.bat` で起動（自動掃除）
